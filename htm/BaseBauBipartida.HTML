Descrição:

A Base Bau Ortobom Cori é ideal para quem precisa de mais espaço. O produto foi desenvolvido utilizando madeira de reflorestamento, que é ecologicamente correta, e proporciona maior durabilidade e resistência à sua estrutura. A Base Bau Ortobom Cori possui um sistema de amortecimento pneumático que evita o esforço na hora de abri-lo, alça de apoio para o colchão não se deslocar quando você abrir o box e pés rosqueáveis que se ajustam aos desníveis do piso.
